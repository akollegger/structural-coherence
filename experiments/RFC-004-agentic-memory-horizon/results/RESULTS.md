## RFC-001 §8.6 Prediction (for reference)

> For an agent performing tasks of increasing horizon, coherence is maintained without
> added structure so long as the relevant memory fits within the context window. Past the
> point where memory must be externalized and retrieved (the context window as a natural
> $V_c$), coherence degrades unless structure is added, manifesting as self-contradiction,
> action on superseded information, or lost constraints — specifically at horizon points
> where earlier context has scrolled out of the window. The benefit of memory structure is
> negligible below $V_c$ and grows with horizon above it.
